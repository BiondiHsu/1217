# Streamlit App – Cloud-first Version

This is a cloud-deployed Streamlit application.

The app is designed to run on **Streamlit Cloud**.
Local execution using `streamlit run` is **not required** and may not work in restricted system environments.

## How to Run
The application is deployed via Streamlit Cloud.
No local setup is needed.

## Files
- `app.py` – main Streamlit application
- `requirements.txt` – Python dependencies

## Notes
This project follows a **cloud-first deployment strategy** to ensure compatibility across different systems.
